# 📘 HTML Implementation Practice Questions

This document contains **practical implementation questions** from **Basic to Advanced level** for mastering HTML.

---

# 🟢 LEVEL 1 – BASIC IMPLEMENTATION

---

## 1️⃣ Create Your First HTML Page

Create a webpage that includes:

* Page title
* One heading
* One paragraph

---

## 2️⃣ Create a Profile Card

Build a simple profile section containing:

* Profile Image
* Name (Heading)
* Short Bio (Paragraph)
* Contact Link

---

## 3️⃣ Create a Simple List

* Create an unordered list of 5 fruits
* Create an ordered list of 5 cities

---

## 4️⃣ Create a Hyperlink Page

Create:

* One link that opens in the same tab
* One link that opens in a new tab

---

## 5️⃣ Insert an Image

Display an image with:

* `src`
* `alt`
* Custom width and height

---

## 6️⃣ Create a Basic Table

Create a table with:

* 3 Columns → Name, Age, City
* 3 Rows of data
* Add a caption

---

## 7️⃣ Create a Simple Form

Create a form that includes:

* Name input
* Email input
* Password input
* Submit button
